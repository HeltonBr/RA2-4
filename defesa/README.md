# Kit de Demonstracao da Defesa

Esta pasta reune arquivos prontos para treino da apresentacao.

## Arquivos

- `demo_valido_base.txt`: caso valido principal, derivado de `tests/teste3.txt`.
- `demo_valido_alterado.txt`: caso valido alternativo para mostrar resultado e Assembly diferentes.
- `demo_erro_lexico.txt`: caso invalido para demonstrar erro lexico.
- `demo_erro_sintatico.txt`: caso invalido para demonstrar erro sintatico.
